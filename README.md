<ul>

    <li>Инструменты - Selenium, Pytest, Pytest-Xdist, Allure, logging</li>
    <li>Паттерны проектирования - Page Object Model, Page Factory</li>

<h3>Для API Тестов</h3>
<ul>

</ul>

<h2>Установка и запуск</h2>
<strong>Преустановка репозитории для запуска API</strong>
<ol>
    <li>Установить Docker + Docker Compose</li>
    <li>Клонировать репозиторий - https://github.com/sun6r0/test-service</li>
    <li>Открыл склонированный репозиторий</li>
    <li>В командной строке запустить контейнер - docker-compose up --build -d</li>
</ol>
<ol>
    <li>Клонировать репозиторий</li>
    <li>Настроить переменные окружения в .venv</li>
    <li>Установить зависимости: <code>pip install -r requirements.txt</code></li>
    <li>Запустить тесты с нужными вам параметрами: <code>pytest -v -s</code> - это выдаст подробное описание тестов и все print</li>
    <li>Если вам нужны конкретные тесты, то запускайте так - <code>cd tests</code>, потом <code>pytest *Название файла*</code></li>
    <li>Если нужен отчет allure: <code>pytest --alluredir=allure-results</code></li>
    <li>Далее прописываете <code>allure serve allure-results</code></li>
    <li>Для параллельных тестов: <code>pytest -v -n auto</code></li>
    <li>Если желаете запустить только UI тесты - <code>pytest tests/ui_testCases -v -n auto</code>, а если API - <code>pytest tests/api_testCases -n -v auto</code></li>
</ol>
